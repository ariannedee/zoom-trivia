How it works

Before the game starts

  • Create your team or join an existing team below
  • Join the {% if game.link %} Zoom event {% else %} Zoom event {% endif %} and change your display name to your team name

How the game will work

  • Each round you will be put into breakout rooms while you answer the questions. We will use your display name to help sort you into rooms.
  • In this app, click the Start → button for the current round to go to the question page
  • Only one team member should fill in and submit the questions. They may want to share their screen so that others can see what has been answered.
  • There will be a timer that shows up when you have less than one minute to submit your answers
  • After submitting, you can stay in your breakout room while the round gets marked
  • After marking, you will be brought back into the main group, go over the answers, then start the next round
